# SQL Analytics Contábil com PostgreSQL

Este projeto foi desenvolvido durante meus estudos em SQL para Análise de Dados e Data Science, com foco em PostgreSQL e consultas analíticas aplicadas a dados contábeis.

O objetivo foi praticar:

- modelagem de dados
- agregações SQL
- funções estatísticas
- tratamento de valores nulos
- consultas analíticas
- sumarização e distribuição de dados

Os dados utilizados são simulados e foram gerados automaticamente através de stored procedures em PostgreSQL.


## Tecnologias
- PostgreSQL
- SQL
- pgAdmin

## Funcionalidades
- geração automática de dados
- sumarização financeira
- métricas estatísticas
- análise de distribuição de dados
- cálculo de mediana e quartis

## Conceitos Aplicados
- GROUP BY
- CASE WHEN
- SUM()
- AVG()
- STDDEV()
- PERCENTILE_CONT()
- tratamento de NULL

## Estrutura
- criação de schema e tabela
- procedure de carga
- queries analíticas
- distribuição estatística
